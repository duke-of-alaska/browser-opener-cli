# browser-opener-cli

## What is it?
CLI to open websites on specific browsers within terminal.

## How do I install it?
Run:
```bash
git clone https://github.com/duke-of-alaska/browser-opener-cli.git
```

If you want it in your current directory:
```bash
git clone https://github.com/duke-of-alaska/browser-opener-cli.git .
```
